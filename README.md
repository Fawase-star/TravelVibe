# TravelVibe – Site Web de Voyage  
Projet réalisé dans le cadre du module HTML/CSS.

##  Présentation
TravelVibe est un site web statique présentant différentes destinations de voyage, une galerie d’images, un blog simplifié, une page À propos ainsi qu’une page Contact.  
Le site est entièrement construit en **HTML5** et **CSS3**, sans JavaScript.

##  Pages du site
- **index.html** – Page d’accueil avec image héro et introduction  
- **destinations.html** – Présentation de destinations populaires  
- **galerie.html** – Galerie d’images en grille (3×3)  
- **blog.html** – Articles de blog (mise en page cartes)  
- **apropos.html** – Page d’informations sur TravelVibe  
- **contact.html** – Formulaire de contact

##  Technologies utilisées
- **HTML5** (structure sémantique)
- **CSS3**
  - Flexbox
  - Grid
  - Animations
  - Variables CSS
  - Pseudo-classes : `:hover`, `:nth-child()`
- **Google Fonts**
- **Font Awesome**
- Hébergement via **GitHub Pages**
